Keyboard shortcuts

File

Editing

  • ACCEL-N: Create a new patch.
  • ACCEL-O: Open a patch.
  • ACCEL-S: Save the current patch.
  • ACCEL-W: Close the current patch.
  • ACCEL-C: Copy objects(s).
  • ACCEL-X: Cut objects(s).
  • ACCEL-V: Paste copied/cut objects(s).
  • <Delete>: Delete objects(s).
  • ACCEL-A: Select all patch objects.

Patch

Object

  • ACCEL-E: Toggle patch edit mode.
  • ACCEL-[Click]: Toggle patch edit mode.
  • [Double click]: Create new object.
  • ACCEL-<Shift>-H: Display this Help screen.
  • ACCEL-<Shift>-[Drag]: Resize object.
  • ACCEL-I: Open Inspector Window.
  • ACCEL-L: Hide object(s) when locked.
  • ACCEL-K: Show object(s) when locked.
  • ACCEL-0: Bring object(s) forward.
  • ACCEL-9: Send object(s) back.
